Configuration Guide

IP Whitelist (WHITELISTED_IPS)

List of IP addresses allowed to access the admin panel:

  • • Use string format: '192.168.1.100'
  • • IPv6 supported: '::1'
  • • Always include localhost: '127.0.0.1'
  • • Don't remove your current IP!
Database Settings

PostgreSQL connection settings:

  • • DB_PASSWORD: PostgreSQL password
  • • DB_NAME: Database name (default: snaily-cad-v4)
  • • DB_USER: Database user (default: postgres)
  • • DB_HOST: Database host (default: localhost)
